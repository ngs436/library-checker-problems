問題文
---------

長さ $N$ の数列 $a_0, a_1, ..., a_{N-1}$ に $Q$ 個のクエリが飛んできます。処理してください。

- `$l_i$ $r_i$`: $\min(a_{l_i}, a_{l_i+1}, ..., a_{r_i - 1})$ を出力する。

制約
---------

- $1 \leq N, Q \leq 500,000$
- $0 \leq a_i \leq 10^9$
- $0 \leq l_i < r_i \leq N$

入力
---------

~~~
$N$ $Q$
$a_0$ $a_1$ ... $a_{N - 1}$
$l_1$ $r_1$
$l_2$ $r_2$
:
$l_Q$ $r_Q$
~~~
